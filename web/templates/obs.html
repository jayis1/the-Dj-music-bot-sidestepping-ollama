To control OBS from Mission Control, you need:

1️⃣ Install OBS Studio (v28+ includes WebSocket)
2️⃣ In OBS: Tools → WebSocket Server Settings → Enable
3️⃣ Set a password in OBS WebSocket settings
4️⃣ Add these to your .env file:
OBS_WS_ENABLED=true
OBS_WS_HOST=localhost
OBS_WS_PORT=4455
OBS_WS_PASSWORD=your_password

Then restart the bot and refresh this page.
