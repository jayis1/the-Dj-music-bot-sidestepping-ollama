📄 Example Nginx Config 
server {
    listen 443 ssl http2;
    server_name radio.yourdomain.com;

    ssl_certificate     /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;

    location / {
        proxy_pass http://{{ web_host }}:{{ web_port }};

        # ── Required headers for ProxyFix ──
        proxy_set_header Host              $host;
        proxy_set_header X-Forwarded-For   $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_set_header X-Forwarded-Host  $host;
        proxy_set_header X-Forwarded-Prefix /;

        # ── WebSocket support (for Activity Log SSE) ──
        proxy_http_version 1.1;
        proxy_set_header Upgrade    $http_upgrade;
        proxy_set_header Connection "upgrade";

        # ── Timeouts ──
        proxy_read_timeout 86400s;
        proxy_send_timeout 86400s;
    }
}
🛠️ Nginx Proxy Manager Setup
  1. In NPM, go to Proxy Hosts → Add Proxy Host
  2. Domain: radio.yourdomain.com
  3. Scheme: http
  4. Forward Hostname/IP: {{ web_host }} (your bot's IP)
  5. Forward Port: {{ web_port }}
  6. Enable Block Common Exploits and Websockets Support ✅
  7. Under Advanced tab, add these headers: 
    proxy_set_header X-Forwarded-Proto $scheme;
proxy_set_header X-Forwarded-Host  $host;
  8. Enable REVERSE_PROXY=true in your .env and restart the bot

⚠️ Danger Zone

These actions will affect the bot process. Restart will reboot the bot; shutdown will stop it completely.
